Bitcoin CSV Parser

exit X

This was a project to take a .csv file and parse that data into .json data. I have the backend built in Node.js and the requests handled by express. On the front end I made the visualization in THREE.js, all the requests are handled with jQuery.

  • Node.js
  • Express
  • Insomnia
  • npm
  • parcel.js
  • THREE.js
  • WebGL
  • GLSL
  • JavaScript
  • HTML5
  • SCSS

The process for the backend was first to parse the csv file. I used an npm package called csv-parser which returns an array of .json objects. In hind site it would have been better to build the csv parser manually. I did do this in JavaScript local storage, I brought the csv file in as a string and then parsed that string with the .split() and then parsed 1 line each milisecond, using that new data to update the size of the next bar on the graph.

The front end is a WebGL scene created using the THREE.js library. Displayed is a grid of boxes created by getting the ammount of data displayed and taking its closest square root. I then loop through and change every boxes x position by 2, once the number of boxes equals the closest square the x position goes back to 0 and the z index increases by 2. This continues until the number of rows in the z column is also equal to the square and the loop ends.
